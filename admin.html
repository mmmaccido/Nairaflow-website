Push your website to GitHub
Step-by-step guide — no prior experience needed
  • 1

    Create a GitHub account

    Go to github.com and sign up for a free account if you don't have one. Use your business email.

  • 2

    Install Git on your computer

    Download Git from git-scm.com. After installing, open your Terminal (Mac/Linux) or Command Prompt (Windows) and confirm it works:

    git --version # Should print something like: git version 2.40.0
  • 3

    Create a new repository on GitHub

    Log into GitHub → click the green New button → name your repo (e.g. nairaflow-website) → set it to Private → click Create repository.

  • 4

    Set up your project folder

    Create a folder on your computer called nairaflow-website. Put both index.html and admin.html inside it. Then open Terminal/Command Prompt inside that folder and run:

    git init git add . git commit -m "Initial website upload"
  • 5

    Connect and push to GitHub

    Copy the repo URL from GitHub (looks like https://github.com/yourname/nairaflow-website.git), then run:

    git remote add origin https://github.com/yourname/nairaflow-website.git git branch -M main git push -u origin main # GitHub will ask for your username and password the first time. # Use your GitHub username and a Personal Access Token as the password. # Create a token at: github.com → Settings → Developer Settings → Tokens
  • 6

    Every time you update the site

    After making changes and downloading the updated index.html from this dashboard, replace the file in your folder and run:

    git add . git commit -m "Update rates and testimonials" git push
  • 7

    Make it live with GitHub Pages (free hosting)

    On your GitHub repo page → click Settings → scroll to Pages → under Source select Deploy from branch: main → click Save. Your website will be live at:

    https://yourname.github.io/nairaflow-website

    For a custom domain like nairaflow.com, add your domain in the Pages settings and update your domain's DNS records to point to GitHub.

Pro tip: VS Code makes this much easier

Download VS Code (free). It has a built-in Git panel — you can click buttons instead of typing commands. Open your project folder in VS Code, click the branch icon on the left sidebar, and it will walk you through committing and pushing without any Terminal commands.
